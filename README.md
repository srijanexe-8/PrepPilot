# 🚀 PrepPilot

Welcome to **PrepPilot**, your ultimate co-pilot for exam preparation, interview readiness, and structured study planning. Whether you're gearing up for technical interviews, standardized tests, or certifications, PrepPilot helps you navigate your prep journey with data-driven insights and interactive study tools.

---

## ✨ Features

- **🎯 Personalized Study Paths**: Dynamically generated schedules tailored to your target exam date and daily study availability.
- **📝 Interactive Practice Arena**: A rich library of flashcards, custom quizzes, and mock tests with instant feedback.
- **📊 Analytics Dashboard**: Track your strengths and weaknesses over time with insightful progress charts.
- **⏰ Smart Reminders**: Integrated notification systems to keep you consistent and on track.

---

## 🛠️ Tech Stack

PrepPilot is built using a modern, scalable web stack:

- **Frontend**: Next.js / React, TailwindCSS, Lucide Icons
- **State Management**: Zustand / React Context
- **Database / Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Analytics**: Chart.js / Recharts

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js (v18+) and npm installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/gitghost66/PrepPilot.git
   cd PrepPilot
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the app in action!
